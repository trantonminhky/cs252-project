(async) generateReviews(place, modelopt) → {Promise.<ServiceResponse>}

Description:

  • Service function for /api/ai/generate-reviews. Generate generic reviews for a location (lmao). This method is mainly used in the frontend to generate fake reviews, as the date field is Flutter formatted. Supports POST requests.
