(async) refresh(sessionToken) → {Promise.<ServiceResponse>}

Description:

  • Service function for /api/profile/refresh. Refreshes the user's access token. Will always fail if staySignedIn was not checked during the login phase. The frontend is thus expected to then prompts the user to login again. Supports POST requests.
